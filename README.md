AI-powered hiring assistant chatbot built using Streamlit and GPT-based LLMs for initial candidate screening.
## Features
- Structured candidate data collection
- Tech-based dynamic question generation
- Context-aware conversation
- Exit handling
- Input validation
- Secure JSON storage
## Installation
1. Clone repository
2. Install dependencies:
   pip install -r requirements.txt
3. Add .env file with OpenAI API key
4. Run:
   streamlit run app.py
## Prompt Engineering Strategy
- Role-based system prompting
- Structured tech-stack injection
- Controlled output generation
- Context-aware message handling
## Data Handling
- Local JSON storage
- No API keys stored in code
- Basic GDPR-compliant handling
